# 📚 Curso JavaScript — SENAI Cotia

Repositório de estudos do curso de **Desenvolvimento JavaScript** do SENAI Cotia.  
Contém anotações, introduções e exercícios práticos desenvolvidos ao longo do curso.

---

## 🧠 Conteúdo Estudado

| Módulo | Tópicos |
|--------|---------|
| HTML | Estrutura, listas, links, formulários, imagens |
| CSS | Box model, flexbox, estilização, responsividade |
| JavaScript | Variáveis, operadores, condicionais (`if/else`, `switch`) |
| JavaScript | Laços de repetição (`while`, `do while`, `for`) |
| JavaScript | Funções, parâmetros, retorno, funções anônimas |
| JavaScript | Objetos, propriedades, métodos, `this`, funções construtoras |

---

## ✅ Exercícios

A pasta `exercicios do curso/lista de exercicios/` contém exercícios com progressão gradual de dificuldade.

Para rodar os exercícios, abra o `index.html` dessa pasta no navegador — ele exibe uma interface para selecionar e executar cada arquivo `.js` individualmente.
